function computePayment(a, r, y, n) {
    var i = 1;
    while (i <= y) {
        var p = (a * r) / (1 - (Math.pow((1 + r), n)));



        // document.getElementById('output').innerHTML = text;
        i++;
    }
    return p;
}

function computeBalance(a, r, y, n, p) {
    var p = computePayment(a, r, y, n)
    var i = 1;
    while (i <= y) {
        var f = (a * (Math.pow((1 + r), n))) - ((p * ((Math.pow(1 + r, n)) - 1))) / r;

Conditionals

In this Sandbox, I use two different conditionals, a switch and an "if/else" The if else is a common method of validating a conditional booleon. If the statement is true, the code in the "if" block will execute. If the the statement is false the code in the "else" block will execute. when you have multilple options to evaluate, a Switch is a more efficent method than nested "if/else" statements this cases compares multiple values and performs an action depending on the true value.


Loops

There are three types of loops used in this example; while, for and do while. each of these loops can provide almost identical functionality, with a few differences. The while loop is one of the most common, and the for loop is a more condensed version. The do while loop is similar to the previous two, except that it checks it's condition at the end of each loop execution. With a do while loop, the loop will always execute at least once until the condition is evaluated. a while loop or for loop validates before the loop runs



Variables

variables are the most simple objects in JavaScript. They can contain a single value, an array or an associative array (also an object). varriables are declared with "var" "name" = "value";


Functions and Parameters

Functions defined in JavaScript execute a block of code and must be "called" by an action such as click of a button. A function may exevcute a block of code, and then send variables, called parameters to another function by calling that function. The arguments passed in the parinthesis of the function call are refered to as the parameters.


Arrays

A Standard array is a variable capable of storeing multiple values. They are declared the same way as a variable, using bracket notation to store and access the multiple values. An associative array is actually a full javascript object. Unstead of using just numbers to access the properties of the array, you use a name or "key". These object properties can also be access with bracket notation, however dot notation is more common.
